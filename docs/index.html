Introduction

The Founding of Penn State Behrend Project Will Stiller, Milo Witcher, and Jax Abele is a collaborative final project for the Digit 110 class. The goal with this project is to take a number archived files dating back to the establishment of the university and make them readily available and organized on a website for people to easily view and analyze. This project contains a gallery with several archived files where users can choose a document and be presented with the original paper followed by a reading view.
